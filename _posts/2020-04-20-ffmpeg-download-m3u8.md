## 下载转格式

如果有指向m3u8的链接，则有最为简单的下载方式：

```bash
ffmpeg -i [视频链接] [保存文件]
```

例如，我需要保存抓包下来的网课至`test.mp4`，则进行命令

```bash
ffmpeg -i 'http://link.to.your/video.m3u8' test.mp4
```

运行效果如下：

![shell](shell.png)

然后就可以在目标目录找到对应文件了

## 关于m3u8的问题

m3u8中ts文件的链接有相对与绝对之分，相对地址只能运用链接下载，绝对地址可以以本地的m3u8文件进行下载。
